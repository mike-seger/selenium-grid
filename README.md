# selenium-grid

Running a Selenium Grid with Firefox and Chrome browsers and drivers in 2
minutes.  

 
# Using

```
# Start grid with 1 node each
$ docker-compose up

# Start grid with 15 nodes each
$ docker-compose scale chrome=15 firefox=15

# Stop grid
$ docker-compose down
```
